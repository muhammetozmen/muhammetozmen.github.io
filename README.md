# Kişisel Web Sitem

Bu proje, [Jekyll](https://jekyllrb.com/) kullanılarak hazırlanmış kişisel web sitemdir.  
GitHub Pages üzerinde host edilmektedir: [https://muhammetozmen.org](https://muhammetozmen.org)

## Hakkında
Sitenin amacı, yaptığım çalışmaları, projeleri ve yazılarımı düzenli bir biçimde paylaşmaktır.  
Basit, açık kaynaklı ve geliştirilmeye açık bir yapıdadır.

## Lisans
Projeyi kendi GitHub hesabınıza forklayabilir veya doğrudan kopyalayarak kendi kişisel siteniz için kullanabilirsiniz.  
Bu proje açık kaynaklıdır, istediğiniz şekilde düzenleyip kullanabilirsiniz.
