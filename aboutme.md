## Technical Me
I develop C/C++ on STM32/ARM Cortex-M in bare-metal and Linux user space. I write drivers based on UART/I2C/SPI, ADC/PWM, DMA, and interrupts; build desktop interfaces with Qt, and develop on-device AI models with OpenCV and TFLite Micro. I also have IoT and LLM-based projects.

## Skills

- Bare-metal programming with Rust, C/C++, and Embedded C
- UART, GPIO, I2C protocols on STM32 and Raspberry Pi
- Linux headless mode, bash terminal
- Python (NumPy, OpenCV, TensorFlow, etc.)
- Computer vision with YOLO, Haar Cascades
- Database management with SQLite and MySQL
- Qt GUI development, MVC compliance
- IoT systems and AI integration
